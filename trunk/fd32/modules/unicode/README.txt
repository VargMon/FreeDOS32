This library provides facilities to manage Unicode (or ISO-10646) character
strings encoded in UTF-8 and UTF-16 and wide characters (format conversion,
case folding).
This library has been developed as part of the FreeDOS-32 project:
   http://freedos-32.sourceforge.net

Documentation for the library is available in the source files in Doxygen
format. You can generate on-line documentation, such as HTML, using Doxygen.
